## Préprocesseur

### Macro 

Remplacement de chaîne simple : 

```c
#define NOM CONTENU
#define PI 3.14
#define U Les choux sont blancs
```

Macros avec paramètres: 

```c
#define MAX(a, b) (a > b ? a : b)
MAX(23, 42)
```

/!\ Attention

1. Toujours protéger les macros avec des parenthèses
2. Les macros sont du REMPLACEMENT DE CHAINES

```c
#include 
#define
#if
#else
#elif
#endif
#pragma
# 
## 
```
